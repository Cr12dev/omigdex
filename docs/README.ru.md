# Omigdex

Приложение для скачивания видео с открытым исходным кодом, созданное с помощью Tauri, React, TypeScript и Rust. Легко скачивайте видео с YouTube, Instagram, TikTok и Pinterest.

## Возможности

- **Поддержка нескольких платформ**: Скачивайте видео с YouTube, Instagram, TikTok и Pinterest
- **Несколько форматов**: Скачивайте в форматах MP4 (видео) или MP3 (аудио)
- **Выбор качества**: Выбирайте между лучшим, высоким (1080p), средним (720p) или низким качеством
- **Очередь загрузок**: Автоматическое управление до 3 одновременных загрузок
- **История загрузок**: Отслеживайте все свои загрузки с постоянным хранением
- **Светлая/тёмная тема**: Переключение между светлым и тёмным режимами
- **Уведомления toast**: Отзывы в реальном времени о статусе загрузки
- **Минималистичный интерфейс**: Чистый и современный дизайн без эмодзи
- **Переносимая версия**: Запуск без установки (включает yt-dlp)
- **Быстрые загрузки**: Оптимизировано для скорости (15-минутное видео за ~10 секунд)

## Технологический стек

- **Frontend**: React + TypeScript + Vite
- **Backend**: Rust с Tauri
- **Стили**: Tailwind CSS + shadcn/ui
- **Загрузчик видео**: yt-dlp (встроенный)
- **Система сборки**: NSIS (установщик) + GitHub Actions

## Установка

### Из Release

1. Скачайте последнюю версию с [GitHub Releases](https://github.com/Cr12dev/omigdex/releases)
2. Выберите:
   - **Установщик (NSIS)**: Запустите установщик `.exe`
   - **Переносимая**: Распакуйте `.zip` и запустите `omigdex-app.exe`

### Из исходного кода

**Предварительные требования**:
- Node.js (v18 или выше)
- pnpm
- Rust toolchain
- yt-dlp (для разработки)

**Шаги**:

```bash
# Клонируйте репозиторий
git clone https://github.com/Cr12dev/omigdex.git
cd omigdex

# Установите зависимости
pnpm install

# Запустите в режиме разработки
pnpm tauri dev

# Соберите для производства
pnpm tauri build
```

## Использование

1. **Скачать видео**:
   - Вставьте URL видео в поле ввода
   - Выберите формат (MP4 или MP3)
   - Выберите качество (Лучшее, Высокое, Среднее, Низкое)
   - Нажмите "Download"

2. **Управление загрузками**:
   - Просмотр активных загрузок на вкладке "Queue"
   - Отмена загрузок при необходимости
   - Очистка завершённых загрузок

3. **Просмотр истории**:
   - Переключитесь на вкладку "History"
   - Просмотр всех прошлых загрузок
   - Удаление отдельных записей или очистка всех

4. **Переключение темы**:
   - Нажмите кнопку темы в заголовке
   - Переключение между светлым и тёмным режимами

## Архитектура Rust Backend

Backend состоит из нескольких модулей:

- **`main.rs`**: Точка входа приложения
- **`lib.rs`**: Определения команд Tauri и инициализация
- **`downloader.rs`**: Интеграция с yt-dlp для загрузки видео
- **`queue.rs`**: Очередь загрузок с параллельным управлением
- **`history.rs`**: Постоянная история загрузок в JSON
- **`types.rs`**: Структуры данных и перечисления

## Известные проблемы

- **Предупреждения антивируса**: Неподписанные исполняемые файлы могут вызвать Windows Defender или другое антивирусное программное обеспечение. Это нормально для неподписанных приложений. Рассмотрите возможность добавления исключения или использования переносимой версии.
- **Подписывание кода**: Чтобы избежать предупреждений антивируса, приложению требуется цифровая подпись. Для проектов с открытым исходным кодом [SignPath.io](https://signpath.io/) предлагает бесплатное подписание кода.
